# usage


각 리스트에서 원하는 원하는 항목을 url로 접근.

docker example.

```
curl https://juhyun-nam.github.io/dev/docker
```

brew example.

```
curl https://juhyun-nam.github.io/dev/brew
```

바로 실행할 때는

```
curl https://juhyun-nam.github.io/dev/docker | bash
```
